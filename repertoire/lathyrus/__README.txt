The required sound files are not tracked by git. Instead, download an archive
of the "sounding" folder, and unzip it into the "lathyrus" folder. So the
"lathyrus" folder will contain the "sounding" folder, which contains all the
sound files.

Download the archive of the "sounding" folder here:

https://iu.box.com/s/yagrugfzbeuhzr8xch2s4xccxvi9btqe
