The required sound files are not tracked by git. Instead, download an archive
of the "sound" folder, and unzip it into the "memory" folder. So the "memory"
folder will contain the "sound" folder, which contains all the sound files.

Download the archive of the sound folder here:

https://iu.box.com/s/pr70sb6lu0xnfgezo9z2v58q0va1bn9q
