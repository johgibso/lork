The required sound files are not tracked by git. Instead, download an archive
of the "sound" folder, and unzip it into the "why-is-it-dancing" folder.
So the "why-is-it-dancing" folder will contain the "sound" folder, which
contains all the sound files.

Download the archive of the sound folder here:

https://iu.box.com/s/ra3vyy9j7niyca9mqe2eh2hk5n2ev8km
